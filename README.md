# minstrel-4th-roms
ROMs for the Minstrel 4th

* https://www.tindie.com/products/tynemouth/minstrel-4th-z80-forth-rc2014-jupiter-ace-kit/
* https://z80kits.com/shop/tynemouth-minstrel-4th/

The Minstrel 4th can support 8K ROM images from $0000-$1FFF, and also has the option for 16K ROMs from $0000-$3FFF, although code is only accessible from $0000-$1FFF and $2800-3BFF as the other ranges are used for the video RAM and 1K base system RAM.

Current images for the Minstrel 4th include:

* The standard Jupiter ACE Forth ROM
* Original Enhanced ROM
  * https://github.com/nihirash/Minstrel4th-rom-patch
* Minstrel 4th ROM (updated Enhanced ROM - work in progress)
  * [https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/Minstrel 4th](https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/Minstrel%204th)
* ZX80 4K Integer BASIC
  * [https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/4K BASIC](https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/4K%20BASIC)
* ZX81 8K Floating Point BASIC
  * [https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/8K BASIC](https://github.com/tynemouthsoftware/minstrel-4th-roms/tree/main/8K%20BASIC)
* Micro Professor MPF-1
  * https://github.com/markgbeckett/jupiter_ace/tree/master/mpf-i
* ZX Forth (work in progress)
  * https://github.com/markgbeckett/zx81/tree/main/zx-forth
